Procédure d'installation temporaire

Préparation
===========

1. Se placer dans le répertoire LivraisonsNabLab
2. Créer un répertoire NabLab-V<date> (date anglaise pour le tri)
3. Changer le lien last pour qu'il pointe vers ce répertoire


Déploiement Eclipse
===================

1. Lancer Eclipse de développement et aller dans le projet fr.cea.nabla.feature.
2. Editer feature.xml et cliquer sur Export Wizard (à droite). 
3. Choisir comme destination : NabLab-V<date>/EclipseFeature


Déploiement des bibliothèques
=============================

1. Se placer dans le workspace de développement
2. cp -R fr.cea.nabla.cpplib ~/LivraisonsNabLab/last
3. cp -R fr.cea.nabla.javalib ~/LivraisonsNabLab/last


Sur le poste utilisateur
========================

Le poste utilisateur doit comporter un répertoire NabLab contenant :
- un répertoire workspace contenant les exemples (mais pas les tests)
- un répertoire par livraison NabLab (le workspace n'est pas relivré)
- un lien last permettant d'éviter de changer les chemins de compilation et de lib

1. Copier le répertoire NabLab-V<date> dans NabLab
2. Mettre à jour le lien last
3. Lancer Eclipse utilisateur
4. Help > Install New Software > Add > Local et choisir le répertoire NabLab-V<date>
   Attention : décocher 'Groups items by category'
5. Suivez les instructions et redémarrer Eclipse.
